## Technical Specifications

**Frontend:** React 18, TypeScript, Vite, Tailwind CSS
**Backend:** Flask 3.0, Python 3.11
**AI Engine:** Claude AI (Opus 4.5)
**Data Source:** booking.com via MCP (Model Context Protocol)
**Architecture:** Full-stack web application with RESTful API
**Deployment:** Express.js proxy server
**Compatibility:** All modern browsers, responsive design
