## Technology Stack

**Frontend:**
- React 18 with TypeScript

- Responsive design principles

**Backend:**
- Flask 3.0 (Python)

- Express.js proxy server

## How It Works

1. **Type Your Request** - Use natural language to describe your travel needs
2. **AI Processing** - Claude AI understands your intent and searches for flights
3. **Real-Time Data** - MCP tools fetch live flight information from booking.com
4. **Beautiful Results** - See all options displayed in elegant flight cards
5. **Make Your Choice** - Compare prices, times, and airlines at a glance

## Getting Started

```bash
# Clone the repository
git clone https://github.com/NinjaTech-AI/agent-jetset-ai.git

# Install dependencies
cd agent-jetset-ai
npm install
cd frontend && npm install
cd ../backend && pip install -r requirements.txt

# Start the application
./start.sh
```

Visit http://localhost:9000 to start searching for flights!
